# Quick Start Guide

## Get Started in 3 Steps

### 1. Navigate to Project
```bash
cd cannyminds-landing
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Open Browser
Visit [http://localhost:3000](http://localhost:3000)

## What You'll See

A stunning landing page featuring:
- Smooth scroll animations
- Parallax effects
- Interactive sections
- Beautiful color scheme
- Professional design

## Key Features

### SEO Optimized
- Google E-E-A-T compliant
- Schema.org structured data
- Proper meta tags
- Mobile-friendly

### Performance
- Next.js 15 with App Router
- Optimized images
- Code splitting
- Fast load times

### Animations
- Framer Motion scroll effects
- Lenis smooth scrolling
- Reveal on scroll
- Parallax backgrounds

## Sections

1. **Hero** - Engaging introduction with CTAs
2. **Products** - CannyDocs, CannyHR, CannyScan, CannyTrack
3. **Services** - IT solutions overview
4. **About** - E-E-A-T credentials
5. **Global Offices** - 4 worldwide locations
6. **Contact** - Form with validation

## Customization

### Change Colors
Edit [tailwind.config.ts](tailwind.config.ts)

### Update Content
Edit components in [components/sections/](components/sections/)

### Modify SEO
Edit [lib/seo.ts](lib/seo.ts)

## Build for Production

```bash
npm run build
npm start
```

## Deploy

Recommended: [Vercel](https://vercel.com)

```bash
vercel
```
